# Logo Pack

Files:
- `assets/logo/vrs-logo-mark.svg` - primary square brand mark (app icon, social image base, avatar).
- `assets/logo/vrs-logo-lockup.svg` - horizontal logo with wordmark (header, docs, GitHub social preview base).
- `assets/logo/vrs-logo-favicon.svg` - simplified small-size mark for favicon/tab icon.

Quick usage:
- Website nav/logo area: use `vrs-logo-lockup.svg` (or just mark on mobile).
- Favicon: use `vrs-logo-favicon.svg`.
- GitHub README image: use `vrs-logo-lockup.svg`.

Notes:
- All logos are pure SVG, no external assets or fonts embedded.
- Colors match the current UI palette (deep blue glass background, court orange, rotation blue, libero red).
